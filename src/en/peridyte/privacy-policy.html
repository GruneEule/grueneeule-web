Privacy Policy for Peridyte

Last update: 09.05.2025

1. What data is collected?

  • Server Data:
    • Server ID, Channel IDs, Role IDs (for ticketing, join roles, etc.)
    • Message IDs (for giveaways, polls, starboard)
    • User IDs (for level system, mod logs, ticketing)
  • Temporary Data:
    • Message cache (for automod, spam detection)

2. Why is data collected?

  • To provide the bot's features (e.g., ticketing system, level system).
  • To comply with server rules (moderation, automod).
  • To improve the bot (analyzing usage patterns).

3. How is data protected?

  • Data is encrypted (JSON) and only used for the intended purpose.
  • No data will be shared with third parties, except when required by law.

4. Data Deletion

  • Individual data can be deleted upon request (via ticket on Discord to the bot operator).
