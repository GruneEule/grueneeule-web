Funktionen

🎫

Ticket-System

Erstelle ein fortgeschrittenes Ticket-System mit benutzerdefinierten Kategorien, Reaktionsrollen und automatischer Archivierung.

🛡️

Moderation

Erweiterte Moderations-Tools wie Warn, Kick, Ban, Mute und automatische Moderation bei Regelverstoß.

🎭

Rollen-Management

Automatische Join-Rollen, Reaktionsrollen und temporäre Rollen für verschiedene Events.

🎁

Giveaway-System

Erstelle und verwalte Gewinnspiele auf deinem Server mit anpassbaren Einstellungen.

🔊

Voice-Channel

Temporäre Voice-Channel die automatisch erstellt und gelöscht werden, wenn sie nicht mehr genutzt werden.

📝

Logging-System

Detaillierte Logs für alle Aktivitäten auf deinem Server, inklusive Nachrichten, Rollen-Änderungen und mehr.

Häufig genutzte Befehle

Ticket-System

Erstelle ein neues Ticket-Panel für deinen Server:

/ticket setup [channel] [category] [support-role]

Schließe ein aktives Ticket:

/ticket close [grund]

Moderation

Warne einen Benutzer:

/warn [user] [grund]

Entferne einen Benutzer vom Server:

/kick [user] [grund]

Rollen-Management

Erstelle ein Reaktionsrollen-Menü:

/roles setup [channel]

Füge eine automatische Join-Rolle hinzu:

/autorole add [role]

Giveaways

Starte ein neues Gewinnspiel:

/giveaway start [zeit] [gewinner] [preis]

Beende ein laufendes Gewinnspiel:

/giveaway end [message-id]

Voice-Channels

Richte temporäre Voice-Channels ein:

/voice setup [category]

Ändere deinen temporären Voice-Channel:

/voice edit [name/limit/permission]

Logging

Richte Logging-Kanäle ein:

/logging setup [channel]

Passe Logging-Einstellungen an:

/logging config [option] [value]

Der Bot wird ständig weiterentwickelt und mit neuen Funktionen ausgestattet, um den wachsenden Anforderungen moderner Discord-Server gerecht zu werden.
